## Run it on Your Machine

- create conda env

```
conda create -n neural-lego python=3.8
```

- activate env

```
conda activate neural-lego
```

- install dependencies

```
pip install -r requirements.txt
```

- run the server

```
cd .\backend\
python manage.py runserver
```

- visit website

```
http://127.0.0.1:8000/
```
